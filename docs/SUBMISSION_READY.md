임팩트: K-엔터테인먼트 $12B+ 시장의 창작자 생산성 10x 향상
```

```
K-WAVE AI STAGE is an interactive creative platform using Gemini as an
AI co-creator for K-drama and K-pop production.

instantly generates K-drama scene scripts, K-pop concept sheets, and story arcs
via real-time streaming.

Core Features:
• K-Drama Scene Generator: Genre/mood/character/conflict → Live dialogue + direction notes

Tech Stack: React + Vite + TypeScript / Cloudflare Workers / Gemini 2.0 Flash API
