## 6. 제품 스펙 (개발팀 배포용)

```
MVP 핵심 기능 (해커톤 데모):
  1. K-드라마 씬 생성기
     Input:  장르(로맨스/스릴러/가족드라마)
             분위기(달달/긴장/슬픔)
             캐릭터 수(2~4명)
             갈등 유형(연애/가족/직장)
     Output: 실시간 스트리밍 대사 + 연출 노트

  2. K-팝 컨셉 시트 생성기
     Input:  그룹 컨셉(걸크러시/청순/4세대 스타일)
             타겟 감정(설렘/자신감/그리움)
             시즌(봄/여름/가을/겨울)
     Output: 그룹 페르소나 + 앨범 컨셉 + 타이틀곡 테마

  3. 스토리 아크 확장기
     Input:  생성된 씬 (자동 컨텍스트 연동)
     Output: 에피소드 1~16화 전체 구조

Tech: React + Vite + TypeScript / Cloudflare Workers / Gemini 2.0 Flash
Repo: https://github.com/CroinDA/kwave-ai-stage-gemini-hackathon
```
